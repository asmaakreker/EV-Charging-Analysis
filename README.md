# EV Charging Cost Analysis

## Overview

This project analyzes electric vehicle (EV) charging session data to determine whether vehicle type affects charging cost.

The dataset contains 500 records, including variables such as vehicle type, energy consumption, charging time, and cost.

---

## Objective

To evaluate the relationship between Vehicle_Type and Cost_INR and determine whether vehicle type significantly influences charging cost.

---

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy

---

## Analysis Performed

* Data cleaning and validation (no missing values)
* Correlation analysis (Pearson and Spearman)
* Pivot table analysis
* Data visualization (boxplots and bar charts)

---

## Key Findings

* Vehicle type has no significant impact on charging cost
* Correlation analysis showed a very weak relationship (r ≈ 0.03)
* Spearman test confirmed no statistical significance (p > 0.05)
* Energy consumption is the main factor influencing cost

---

## Visualizations

The project includes:

* Boxplot comparing cost distribution across vehicle types
* Bar chart showing average cost per vehicle type

---

## Files

* EV-Charging-Analysis.ipynb – Full analysis notebook
* EV-Charging-Analysis.docx – Project report

---

## Conclusion

The analysis shows that charging cost is largely independent of vehicle type and is more strongly influenced by energy consumption.
